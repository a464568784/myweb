#基于PHP、jquery、bootstrap的课程管理系统</br>
##login.html是登录界面；</br>
##admin文件夹是管理员系统；</br>
##teacher文件夹是教师系统；</br>
##student文件夹是学生系统；</br>
##实现了：</br>
      ###管理员界面的管理员操作：</br>
            ####可以添加修改教师、学生、管理员、课程。</br>
      ###教师界面操作：</br>
            ####可以添加修改学生成绩，查看课程，修改自己的密码。</br>
      ###学生界面操作：</br>
             ####可以选课，查看课程成绩，修改自己的密码。</br>

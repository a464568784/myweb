# myweb
基于PHP、jquery、bootstrap的课程管理系统
实现了：
      管理员界面的管理员操作：
            可以添加修改教师、学生、管理员、课程。
      教师界面操作：
            可以添加修改学生成绩，查看课程，修改自己的密码。
      学生界面操作：
             可以选课，查看课程成绩，修改自己的密码。
使用前请先导入数据库system.sql
